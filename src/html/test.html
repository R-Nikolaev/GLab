Спортивное программирование С++

Станьте мастером спортивного программирования на C++. Этот курс поможет вам овладеть навыками решения сложных алгоритмических задач и подготовиться к соревнованиям по программированию. Java Junior разработчик

5 месяцев

По окончании курса вы сможете создавать комплексные приложения на Java, работать с современными библиотеками и технологиями.
